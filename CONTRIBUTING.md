# Contributing

Thank you for contributing to LOT Terminal!

## Quick Start

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/YOUR-USERNAME/lot-terminal.git`
3. **Create a branch**: `git checkout -b feature/your-feature`
4. **Make changes** and test them
5. **Commit**: `git commit -m "Add your feature"`
6. **Push**: `git push origin feature/your-feature`
7. **Open a Pull Request**

## Guidelines

**Code:**
- Follow existing bash style
- Test your changes (`lot init`, `lot help`, etc.)
- Update documentation if needed

**Commits:**
- Use clear, descriptive messages
- Present tense: "Add feature" not "Added feature"

**Pull Requests:**
- Describe what and why
- Link related issues
- Test on Linux or macOS if possible

## Ways to Contribute

- Report bugs
- Suggest features
- Fix issues
- Improve documentation
- Share your hardware projects
- Help others in discussions
