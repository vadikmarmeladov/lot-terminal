## Become a Contributor!

We welcome contributions of all types:

### Code Contributions
- Fix bugs
- Add features
- Improve performance
- Refactor code

### Non-Code Contributions
- Improve documentation
- Create tutorials
- Design graphics
- Test on different platforms
- Answer questions
- Share your projects

### Getting Started

1. Read [CONTRIBUTING.md](CONTRIBUTING.md)
2. Check [open issues](https://github.com/vadikmarmeladov/lot-terminal/issues)
3. Join [discussions](https://github.com/vadikmarmeladov/lot-terminal/discussions)
4. Fork, code, and submit a PR!

## Recognition

Contributors are recognized through:
- This file (CONTRIBUTORS.md)
- GitHub's contributor graph
- Release notes
- Special mentions for significant contributions

**Every contribution matters!** Whether you fix a typo, answer a question, or add a major feature, you're helping make LOT Terminal better for everyone.
